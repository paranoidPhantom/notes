## Кольца и поля

$\ang{\overbracket{R}^{\text{Ring}}, +, *}$ - кольцо ***если***:
1. По сложению - абелева группа
2. $\forall x,y \in R \qquad x(y + z) =  xy + xz \qquad (x+y)z = xz+yz$ - дистрибутивность

### Следствия
1. $\forall x \in R \qquad 0x = x0 = 0$
	   $xy = x(y + 0) = xy + x0$
	   $\cancel{xy} = x(y + 0) = \cancel{xy} + x0$
	   $0 = x0$
2. $\forall x, y \in R \qquad x(-y) = (-x)y = -xy \qquad 0 = x0 = x(y + (-y)) = xy + x(-y)$
	   $\ra 0 = xy + x(-y)$
	   $\ra x(-y) = -xy$
3. $\forall x,y,z \in R \qquad x(y-z)=xy-xz \qquad (x-y)z=xz -yz$

### Прямое произведение колец

$R, Q$ - кольца

$R \times Q$ - $\text{носитель}$ $R \oplus Q$
$(a, b) + (c, d) = (a +_R c, b+_Qd)$ - покомпонентное сложение колец
$(a, b) (c, d) = (a *_R c, b*_Qd)$ - покомпонентное умножение колец

$Q \leq R \quad Q$ - $\text{подкольцо}$ $R: \qquad Q \subseteq R$
1. $Q^+$ - $\text{подгруппа}$ $R^+$
2. $\forall x,y \in Q \qquad xy \in Q$ - замкнутость относительно умножения
Кольцо ассоциативно относительно умножения $\ra$ ассоциативное кольцо
Кольцо коммутативно относительно умножения $\ra$  коммутативное кольцо
Есть нейтральный элемент по умножению $\ra$ называется "кольцом с единицей"

## Определение поля

*Коммутативное и ассоциативное кольцо с единицей в котором каждый ненулевой элемент обратим - называется полем*

#### Примеры
1. $\mb Q, \mb R$
2. $p$ - простое $\lra \mb Z_p$ - поле

## Лемма

В поле нет делителей нуля

##### Доказательство:
$a,b \neq 0 \qquad ab = 0$
$\qquad\qquad\quad a^{-1}ab=a^{-1}0$
$\qquad\qquad\quad b = 0$ противоречие

#### Примеры
1. $\mb R[\epsilon]\quad b\neq 0 \quad b\epsilon \neq 0 \quad (b\epsilon)^2 = 0$ - не поле
2. $\mb R[\omega] \quad (\underbracket {a, a}_{\neq 1})(a,-a) = (0,0) = 0$ - не поле

## Подполе

$F^* = F \setminus \br 0$
$K \subseteq F$
$K$ - $\text{подполе}$ $F \quad K \leq F$
1. $K$ - $\text{подкольцо}$ $F$
2. $1 \in K$
3. $0 \neq x \in K \ra x ^{-1} \in K$

$\mb Q \leq \mb R$
$\mb Q \leq \underbracket{...}_{\text{много всего}} \leq \mb R$
$\mb Q[\sqrt 2] = \br{a + b\sqrt 2 \;\large |\; a,b \in \mb Q}$ - поле

##### Доказательство:
$\frac 1 {a + b \sqrt 2} = \frac{a - b \sqrt 2}{a^2 - 2b^2} = \frac a{a^2 - 2b^2} - \frac b {a^2-2b^2}*\sqrt 2$
$a^2 + b^2 \neq 0 \qquad a^2 - 2b^2 = 0 \qquad \frac a b = \pm \sqrt 2 \notin \mb Q$

## Гомоморфизм колец

$R, Q$ - кольца
$\phi : R \rightarrow Q$ - гомоморфизм колец, если:
$\forall x, y \in R \qquad \phi(x + y) = \phi(x) + \phi(y)$
$\qquad\qquad\qquad\phi(xy) = \phi(x) * \phi(y)$

#### Примеры
1. $\phi : x \mapsto 0$
2. $R =$ кольцо сходящихся последовательностей: $lim: (a_i) \mapsto \underset{i \rightarrow \infty}{lim}\;a_i$
   $Q = \mb R \qquad \underset{i \rightarrow \infty}{lim}(a_i+b_i) = \underset{i \rightarrow \infty}{lim}\;a_i + \underset{i \rightarrow \infty}{lim}\;b_i$
$\chi: P(M) \rightarrow \br{0,1}^M$
$A \subseteq M \quad \chi(x) = \begin{Bmatrix}0, x\notin A\\1,x\in A\end{Bmatrix}$
$\chi_A * \chi_B$
$\chi_A + \chi_B$

3. $\ang{P(M), \Delta, \cap} \simeq \ang{\br{0, 1}^M, +, *}$
   $\chi(A \Delta B) = \chi_A + \chi_B$
   $\chi(A\cap B) = \chi_A * \chi_B$
   $Изоморфизм$ 🥳
   $P(M) \simeq \underbracket{\mb Z_2 \oplus \mb Z_2 \oplus ... \oplus \mb Z_2}_{|M|}$

## Теорема

Если $R$ - коммутативное кольцо - новое кольцо тоже коммутативно;
Если $R$ - кольцо с единицей - новое кольцо тоже с единицей;

1. $R$ - коммутативно
	$c_i = \argsum i {j = 0} a_j\;b_{i-j} = \argsum i {j = 0} b_j\;a_{i-j}$

	$a(bc) = f \quad f_n=\argsum{}{k+j=n}a_k(\argsum{}{j=i+l}b_ic_l)=\argsum{}{k+i+l=n}a_kb_ic_l$
	$d_n=f_n$

## Степень многочленов

## Лемма (о степенях)

$\forall f,x \in R[x] \qquad deg(f+g) \leq max(deg\;f, deg\;g)$

$\forall f,g \in R[x] \qquad deg(fg) \leq deg\;f + deg\;g$
