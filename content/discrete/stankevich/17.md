# Цепи Маркова

![Граф](/_media/Pasted%20image%2020260301143928.png)

Граф поведения Пети, у которого есть честная монета, и который хочет пойти в кино с вероятностью $\frac 13$

Если занумеровать каждое состояние (числом от $1$ до $n$), можно рассмотреть рассмотрение случайной величины:
$b = (b_1,b_2,\dots,b_n)$
$b_i$ - вероятность находиться в $i$-м состоянии.
$c = (c_1,c_2,\dots,c_n) \quad C$

Рассмотрим матрицу $P$ такую, что $p_{ij}$ - вероятность перейти из $i$ в $j$:
$\tmatrix{0 &\frac 12 &\frac 12 &0 \\ 0 &1 &0 &0 &\text{поглощающее} \\ \frac 12 &0 &0 &\frac 12 \\ 0 &0 &0 &1 &\text{поглощающее}}$

$c_i = P(c=i) = \argsum n {j=1}P(C=i|B=j)P(B=j) = \argsum n {j=1}p_{ji}b_j$
$\vec c = \vec b P$

Тогда умножая начальное состояние $b_0$ на матрицу перехода будем получать вероятностные распределения для каждого броска монеты:

$b^0 = (1,0,0,0)$
$b^1 = (0,\frac 12, \frac 12, 0)$
$b^2 = (\frac 12,\frac 12,0,\frac 14)$
$b^3 = (0,\frac 58,\frac 18,\frac 14)$
$\qquad\qquad \overset \vdots \downarrow$
$\qquad (0,\frac 23, 0, \frac 13)$

Заметим что $b^1 = b^0 \cdot P$ и $b^2 = b^1 \cdot P = b^0 \cdot P^2$
Обобщим: $b^n=b^0\cdot P^n$

## Граф цепи Маркова

1) Поглощающие (существенные) состояния
2) Непоглощающие (несущественные) состояния

Цепь Меркова - **поглащающая** $\la \forall$ состояния есть путь до поглощающего состояния

### Эргодический класс

Компоненты сильной связанности графа Марковой цепи (класс эквивалентности на отношении достижимости)
Поглащающий эргодический класс $\la$ нет "выхода"

Эргодическая Маркова цепь $\la$ состоит из одного эргодического класса

Периодический эргодический класс с периодом $d$ $\la$ длина $\forall$ цикла делится на $d > 1$

## Теорема (классификация Марковых цепей)

Любая м.ц. содержит поглащающий эргодический класс

Для непериодического эргодического класса $\exists$ предельное распределение $b: b=bP$
И $\forall b^0 = b^0P^n \rightarrow b$

## Заданы следующие задачи
1) Понять по начальному распределению и вероятности перехода в каком эргодическом классе мы поглотимся
2) Какое стационарное распределение у непериодических эргодических классов Марковских цепей

### Задача о поглощении
Занумеруем состояния в матрице перехода так, чтобы сначала шли непоглащающие, потом поглащающиеся:

![Разбиение матрицы](/_media/Pasted%20image%2020260301145420.png)

Если $i$ и $j$ - непогл.,
$a$ - часть вектора отвечающая за переход в неполг.
Тогда $a^n = a^0Q^n$

Заметим $b^n = b^0P^n = b^{n-1}P$

![Изменение вектора](/_media/Pasted%20image%2020260301221051.png)

Вектор непоглащённого состояния на переходе в другое неполгащённое состояние зависит только от $Q$, т.к. остальное обнуляется.

### Теорема (о поглощении)

Поглощающая Марковская цепь переходит в поглощающее состояние с вероятностью $1$.

$L$ - максимальная длина кратчайшего пути от вершины $i$ до поглащающей.

$X = Q^L$
$x_{ij} = \argsum {}{k_1,k_2,\dots,k_{L-1}} (q_{ik_1}q_{k_1k_2},q_{k_2k_3},\dots,q_{k_{L-1}j})$
Теперь рассмотрим $\argsum{}{j-\text{не поглащающем}}x_{ij} = \argsum {}{k_1,k_2,\dots,k_{L-1},j} (q_{ik_1}q_{k_1k_2},q_{k_2k_3},\dots,q_{k_{L-1}j}) = \delta < 1$
Тогда $\underset{i=1\dots m}{max}\left (\argsum{}{j-\text{не поглащающем}}x_{ij}\right) = max\;\argsum {}{k_1,k_2,\dots,k_{L-1},j} (q_{ik_1}q_{k_1k_2},q_{k_2k_3},\dots,q_{k_{L-1}j}) = \delta < 1$

$n \rightarrow \infty$
$Q^n_{ij} = \left(Q^LQ^{n-1}\right)_{ij} = \argsum {}k Q^L_{ik}Q^{n-1}_{kj} \leq \argsum {}k Q^L_{ik}V_{n-L} \leq \delta V_{n - L}$
Значит вероятность перейти в не поглащающую вершину стремится к нулю
