## Избыточное кодирование

$\text {купи квас пжж} \rightarrow \underbrace{\ov {\underline {\textbf {канал}}}}_{помехи} \rightarrow \text{купи квас псж}$

>*Код постоянной длины куда удобнее для error correction*

$c: \Sigma \rightarrow \mb B^k$
#### Расстояние Хемминга
 
$H(x,y) = \argsum n {i = 1}[1\;if\;x\neq y]$

$c$ обнаруживает  ошибок, если $\forall a \neq b : H(c(a), c(b)) \geq d + 1$

<span style="opacity: 40%">Прим. автора: Расстояние Хемминга полезно рассматривать как кол-во бит которые флипнули, чтобы получить из одной бинарной строки другую</span>

>Пример:
>$H \tmatrix{011011 \\ 010000} = 3$ (кол-во разных бит по $\oplus$)

### Лемма
*Расстояние Хемминга является расстоянием в определении Мат. Анал.*


$c$ исправляет $d$ ошибок, если $\forall a \neq b: H(c(a), c(b)) \geq 2d + 1$
>очев™

### Лемма

$\forall d\; \exists$ код, исправляющий $d$ ошибок

### Теорема

Если $\exists$ код, исправляющий $d$ ошибок для $\Sigma: |\Sigma| = n$, а длина кодового слова $k$, тогда
$2^k \geq n * \argsum d {i = 0} \tmatrix{k \\ i}$

### Следствие

Если $\exists$ код исправляющий одну ошибку для $\Sigma: |\Sigma| = n$, длина кодового слова которого $k$, тогда

$2^k \geq n * (k + 1)$
$\ra n \leq \frac {2^k}{k+1}$

#### Рассмотрим жадный алгоритм

$c_1 \qquad 00 ... 0$
Запрещаем $\matal{\\ y * G(c(c_1), y) \leq 2d \\ S(c(c_1), 2d)}$

$c_2 \qquad x_2$ - не запрещённое
Запрещаем $y: H(x_2, y) \leq 2d$

$c_3 \qquad x_3$ - не запрщённое
$...$
$c_n \qquad x_n$
$\argsum {2d}{i=0}\tmatrix{k\\ i}=S(0,2d)$

### Теорема
Если $(n-1)(\argsum{2d}{i=0}\tmatrix{k \\ i}) < 2^k \quad \ra \exists$ код $(n,k)$ исправляющий $d$ ошибок
