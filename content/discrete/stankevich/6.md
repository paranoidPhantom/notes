>Для почти всех $\frac {2^n} n$ функциональных элементов

При том для всех схем которые известны: $size = O(n)$

---

Возьмём базис $F = \br{\downarrow (NOR)}$

Берём функцию $f(x_1, x_2,...,x_n)$

### $def$ функциональная программа
- Состоит из $t$ строчек
- $i$-ая строка имеет номер $x_{n+1}$
- $x_{n+1} = x_{a_i} \downarrow x_{b_i}$
	  $1 \leq a_i \leq n+i-1$
	  $1 \leq b_i \leq n+i+1$
- $f = x_n + t$

### Теорема

$f$ в $F$ имеет л. п. из $t$ строк $\lra$ имеет схему из ф.э. содержазщую $t$ элементов

$a \land b = (a \downarrow a) \downarrow (b \downarrow b))$
$x_1 \; x_2$
$x_3 = x_1 \downarrow x_1$
$x_4 = x_2 \downarrow x_2$
$x_5 = x_3 \downarrow x_4$

![image](/_media/Pasted%20image%2020251019190151.png)

Заметим что у одной схемы может быть несколько линейных программ
>В схеме выше мы поменяли местами ноды $1$ и $2$


*У разных схем разные линейные программы*

$cnt(t) \leq n * n * (n + 1) * (n + 1) * (n+2) * (n+2) * ... * (n+t-1)*(n+t-1) \leq (n+t)^{2t}$

$c(t) = \argsum t{i=1}cnt(i) \leq t(n+t)^{2t} \leq (n+t)^{2t+1}$

---

$0 < \alpha < 1$
$(n+t)^{2t+1} < \alpha 2^{2^n}$

$t = \frac{c*2^n} n$
$log_2((\frac{c*2^n} n + n)^{\frac{2c*2^n} n + 1}) = (\frac {2c*2^n} n +1)log_2(\frac{c*2^n} n + n) \underbracket{\leq}_{\text{для всех больших n}} \frac{4c*2^n} n log_2(2c*2^n) n \leq \frac{4c*2^n} n log_2\;2^n \leq 4c*2^n$
$log_2(\alpha * 2^{2^n}) = log_2 2^{2^n} + log_2\alpha = 2^n + log_2\alpha$
Если $c < \frac 14, t = \frac{c*2^n} n \quad \forall \alpha \exists n_0: n>n_0,$ то схем размера $\leq t$ достаточно, чтобы задать $\leq \alpha * 2^{2^n}$ функций

$\ra$ для большинства схем нужно хотя бы $\frac{2^n} {4n}$ элементов

### Теорема ($\Omega$ нижней оценке)

$\forall \epsilon \exists n > n_0,$ то схем размера $\leq t$ достаточно, чтобы задать $\leq \epsilon * 2^{2^n}$ функций
